# AP Physics 1 Simulation Lab

An interactive, browser-based physics simulation tool built for students and educators. Designed to complement AP Physics 1 coursework through hands-on virtual experiments.

## What it does

This project provides customizable physics simulations across four core AP Physics 1 units:

- **2D Motion** — Projectile motion, vector components, trajectories
- **Forces** — Newton's laws, free body diagrams, friction, tension
- **Energy** — Kinetic and potential energy, conservation, work
- **Fluids** — Bernoulli's principle, continuity, Pascal's law, buoyancy

Each simulation lets students adjust variables in real time and observe how physical systems respond — bridging the gap between equations and intuition.

## Who it's for

- High school physics students
- AP Physics 1 teachers looking for free, accessible lab tools
- Anyone who learns better by seeing and interacting than by reading
